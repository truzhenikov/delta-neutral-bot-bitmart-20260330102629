# Delta-Neutral Funding Bot

Telegram-бот для дельта-нейтральной стратегии на крипто перп-биржах.

**Стратегия:** открываем лонг на одной бирже и шорт на другой — зарабатываем на разнице funding rate. Рыночный риск нейтральный.

**Поддерживаемые связки:**
- BitMart × Backpack
- Variational × Extended

## 3. Регистрация на биржах

### Backpack
1. Регистрация: [backpack.exchange/join/65d923ff-b224-4dfb-aa6c-cd5346c77651](https://backpack.exchange/join/65d923ff-b224-4dfb-aa6c-cd5346c77651)
2. Пройди KYC
3. Пополни счёт (USDC)
4. Создай API ключ: **Settings → API Keys → Create API Key**
   - Назови как угодно, например `delta-bot`
   - Включи разрешения: `Order` (чтение + торговля)
   - Сохрани `API Key` и `Secret Key`

### BitMart
1. Зарегистрируйся на [BitMart](https://www.bitmart.com)
2. Пополни фьючерсный счёт в `USDT`
3. Создай API ключ с правами на фьючерсную торговлю
4. Сохрани `API Key`, `Secret Key` и `Memo`

### Variational
1. Регистрация: [omni.variational.io](https://omni.variational.io) (подключи EVM-кошелёк)
2. Пополни счёт (USDC)
3. **Как получить vr-token:**
   - Войди на сайт
   - Нажми `F12` → вкладка `Application` → слева `Cookies` → `omni.variational.io`
   - Найди cookie с именем `vr-token` и скопируй значение

> ⚠️ `vr-token` — сессионный токен браузера. Он истекает ровно через **7 дней**. Есть два способа работать с этим:

#### Вариант А — Автоматическое обновление (рекомендуется)

Бот сам обновляет токен через подпись кошелька, когда срок подходит к концу. Тебе ничего не нужно делать руками.

Для этого нужно добавить приватный ключ EVM-кошелька в `.env`:
```
VARIATIONAL_PRIVATE_KEY=0x твой_приватный_ключ
```

**⚠️ Важно:** приватный ключ даёт полный доступ к кошельку. Используй только некритичный кошелёк, на котором нет значимых активов. Ты делаешь это на свой страх и риск — автор не несёт ответственности за любые последствия.

Как получить приватный ключ в MetaMask: три точки у аккаунта → `Account details` → `Export private key`.

#### Вариант Б — Ручное обновление

Оставь `VARIATIONAL_PRIVATE_KEY` пустым. Бот будет присылать предупреждения по нарастающей:
- За 2 дня до истечения — 3 раза в сутки
- За 1 день — каждые 3 часа
- За 6 часов — каждый час

Первое предупреждение закрепится в чате. Когда получишь уведомление — зайди на сайт, скопируй новый `vr-token` и просто **отправь его в чат с ботом**. Бот подхватит его автоматически и открепит предупреждение.

## 5. Установка бота на VPS

### Подключение к серверу

После покупки VPS тебе дадут IP-адрес, логин и пароль. Подключиться к серверу нужно через SSH — это защищённый способ управлять удалённым компьютером через терминал.

**Mac / Linux** — открой Терминал и введи:
```bash
ssh root@ВАШ_IP
```
Введи пароль когда попросит (при вводе символы не отображаются — это нормально).

**Windows** — открой PowerShell (Win+R → powershell) и введи ту же команду. Либо скачай [PuTTY](https://putty.org), введи IP и нажми Connect.

> 💡 Если не получается разобраться — спроси у нейросети (ChatGPT, Claude): «как подключиться к VPS по SSH с Windows/Mac» — объяснит пошагово.

## 6. Настройка .env

Создай файл с настройками на основе примера:
```bash
cp .env.example .env
```
```bash
nano .env
```

Заполни все поля:

```env
# Telegram
TELEGRAM_TOKEN=токен_от_BotFather
TELEGRAM_CHAT_ID=твой_chat_id

# Стратегия
MIN_APR_THRESHOLD=100       # минимальный APR для сигнала (%)
SCAN_INTERVAL_SECONDS=60    # как часто сканировать рынок (секунды)
POSITION_SIZE_USD=50        # размер одной ноги в USD

# Backpack
BACKPACK_API_KEY=           # API Key из настроек Backpack
BACKPACK_API_SECRET=        # Secret Key из настроек Backpack

# BitMart
BITMART_API_KEY=
BITMART_API_SECRET=
BITMART_API_MEMO=

# Variational
VARIATIONAL_TOKEN=          # vr-token из браузера (F12 → Cookies)
VARIATIONAL_WALLET=         # твой EVM адрес (0x...)
VARIATIONAL_PRIVATE_KEY=    # приватный ключ EVM кошелька (для авто-обновления токена, на свой страх и риск)

# Extended
EXTENDED_API_KEY=           # API Key из настроек Extended
EXTENDED_PUBLIC_KEY=        # Public Key из настроек Extended
EXTENDED_PRIVATE_KEY=       # Private Key из настроек Extended
EXTENDED_VAULT_ID=          # Vault ID из настроек Extended
```

Сохрани файл: `Ctrl+O`, затем `Ctrl+X`.

## 8. Автозапуск при перезагрузке сервера

По умолчанию если сервер перезагрузится — бот не запустится сам. Чтобы он запускался автоматически, нужно зарегистрировать его как системный сервис через `systemd` (это встроенный менеджер процессов в Linux).

Создай файл сервиса:
```bash
sudo nano /etc/systemd/system/delta-bot.service
```

Вставь содержимое (замени `ВАШ_ПОЛЬЗОВАТЕЛЬ` на свой логин, например `ubuntu` или `r2d2`):

```ini
[Unit]
Description=Delta-Neutral Funding Bot
After=network.target

[Service]
Type=simple
User=ВАШ_ПОЛЬЗОВАТЕЛЬ
WorkingDirectory=/home/ВАШ_ПОЛЬЗОВАТЕЛЬ/delta-neutral-bot
ExecStart=/home/ВАШ_ПОЛЬЗОВАТЕЛЬ/delta-neutral-bot/venv/bin/python main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Сохрани (`Ctrl+O`, `Ctrl+X`) и активируй:
```bash
sudo systemctl daemon-reload
```
```bash
sudo systemctl enable delta-bot
```
```bash
sudo systemctl start delta-bot
```

Проверить что бот запущен:
```bash
sudo systemctl status delta-bot
```

Посмотреть логи:
```bash
sudo journalctl -u delta-bot -f
```

## 9. Как пользоваться

### Основные кнопки

| Кнопка | Что делает |
|--------|-----------|
| 📊 Мои позиции | Открытые пары: текущий APR, заработок, отклонение цены |
| 🔍 Сканировать сейчас | Ручное сканирование — топ возможностей прямо сейчас |
| 📋 История | Закрытые пары с результатами |
| ⚙️ Настройки | Размер позиции, включение/выключение связок бирж |
| 💙 Поддержать автора | Кошельки для доната |

### Как открыть пару

1. Бот автоматически присылает сигнал когда находит выгодную возможность
2. Или нажми **🔍 Сканировать сейчас** чтобы не ждать
3. В сообщении нажми **✅ Открыть пару** — бот откроет обе ноги автоматически

### Как закрыть пару

1. Нажми **📊 Мои позиции**
2. Найди нужную пару и нажми **❌ Закрыть пару**

### Настройки

- В **⚙️ Настройки** выбери размер позиции для каждой связки
- Там же можно отключить сигналы для ненужной связки (кнопка ✅/❌ рядом с названием)

### Автоматическая защита

Бот закрывает пару автоматически если:
- APR стал отрицательным и держится **больше 4 часов**
- Суммарный APR упал **ниже −50%**
- Цена ушла против **шорт-ноги** более чем на **15%** (предупреждение приходит на 10%)
- Цена ушла против **лонг-ноги** более чем на **15%** (предупреждение приходит на 10%)

## Рекомендации

**Начинай с малого.** Первые позиции открывай на минимальные суммы — $10–15 на ногу. Убедись что бот корректно открывает и закрывает позиции, мониторинг работает, уведомления приходят. Только после этого увеличивай суммы.

**Меньше позиций — лучше.** Каждое открытие и закрытие пары стоит денег: комиссии биржи плюс спред. Лучше держать 1–2 пары на крупные суммы, чем распылять капитал на 5–10 мелких позиций. Математика простая: чем больше сумма на паре, тем быстрее она окупает издержки на вход и начинает реально зарабатывать.
